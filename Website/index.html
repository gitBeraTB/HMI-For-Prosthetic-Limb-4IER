About the Project

What is 4IER?

🧠

Edge AI

Random Forest TinyML classifier running directly on ESP32-S3 with sub-millisecond inference time. No cloud dependency.

💪

3-Channel sEMG

Simultaneous acquisition from biceps, wrist, and grip muscles through high-conductivity gel electrodes.

⚡

Ultra-Low Latency

Complete signal-to-servo pipeline in under 12.5 ms using ESP-NOW peer-to-peer wireless protocol.

🔒

Medical Safety

IEC 60601-1 BF compliant with leakage current below 10 pA through fully isolated battery-powered nodes.

Capabilities

Key Features

01

TinyML on Edge

Random Forest classifier deployed as a C++ header file, executing inference in under 0.4 ms directly on the ESP32-S3 microcontroller.

02

Advanced DSP Pipeline

50 Hz IIR Notch filter removes power-line interference, followed by a 10-tap FIR moving average for signal smoothing.

03

18-Feature Extraction

Sliding window generates RMS, MAV, Waveform Length, Zero-Crossing Rate, SSC, and statistical features every 12.5 ms.

04

ESP-NOW Protocol

Bypasses traditional Bluetooth stack for sub-5ms round-trip wireless communication between nodes.

05

Live GUI Plotter

PyQtGraph real-time visualization with keyboard-driven dataset labeling for efficient data collection workflows.

06

Automated ML Pipeline

Google Colab script with SMOTE balancing and GridSearchCV exports optimized models directly as C++ firmware headers.
